# User Interface Requirements

1. **Login Screen**  
   - Anonymous login using nickname and avatar.

2. **Mood Selection Modal**  
   - Mood selection with emojis.

3. **Chat Screen**  
   - Real-time interface with message bubbles and typing indicators.

4. **Consent Modal**  
   - Requests permission before AI message analysis.

5. **Mood Dashboard**  
   - Displays mood patterns using charts.
